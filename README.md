# Tienda de Muebles - Sitio Web

##  About
Este proyecto es un sitio web para una tienda de muebles, desarrollado con HTML, CSS y JavaScript.  
El objetivo es mostrar productos de manera atractiva, moderna y responsiva para mejorar la experiencia del usuario.

---

##  Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

---

##  Funcionalidades

- Página de inicio con presentación de la tienda
- Catálogo de muebles
- Sección de productos destacados
- Diseño responsive para móviles y tablets
- Animaciones básicas con JavaScript
- Navegación entre secciones

---

##  Características

- Diseño moderno y limpio
- Interfaz intuitiva
- Adaptado a dispositivos móviles
- Estructura simple y fácil de modificar
